 # Homebrew Tap for codoworks

 ## Installation

 \`\`\`bash
 brew tap codoworks/tap
 brew install sshto
 \`\`\`

 ## Available Formulas

 | Formula | Description |
 |---------|-------------|
 | [sshto](https://github.com/codoworks/sshto) | Fast SSH connection manager with interactive TUI |

 ## Updating

 \`\`\`bash
 brew update
 brew upgrade sshto
 \`\`\`
